Conditions


In this chapter we'll go over conditions in Python.

1.1 Boolean

1.1.1 Values

There are two boolean values: True and False.

Boolean values are used for condition purposes.

The values are named after a famous mathematician called George Boole.

You can't save values in True and False . Python will throw an error, if you do so.

However, True and False values can be saved in variables.

1.1.2 Operators

Furthermore there are three boolean operators in Python: and, or, not .

When you are using and both conditions have to be true in order for it to be true.

When you are using or one condition only has to be true in order for it to be true.

Boolean operators are used to compare boolean values with each other.


1.2 Operators

Logical operators evaluate two values and give True or False as an output.

Table 2: Possible logical operators in Python

Logical Operators
> stands for bigger than
< stands for smaller than
<= stands for smaller and equal
>= stands for greater and equal
== stands for equal
!= stands for not equal

1.3 Statements

If-statements: The if-block is executed when the condition is true.

Else-statements: The else-block is executed when the condition in the if-statement is false and the condition in the else-statement is true.

Elif-statement(s): The elif-statement(s) are between the if-statement and the else-statement. The elif-block is only executed when the if-statement is false.

Remember: The block of the first true elif-statement is executed. For that reason the order of the statements are important.

1.4 Loops

While loops: The while loops make sure that the code inside the loop is executed until the condition isn't true anymore.

Now you are probably asking yourself what the difference between if-statements and loops are.

If-Statements are only executed once and don't return to the beginning as while loops do.

For loops: The for loops principally function the same way as while loops. The only difference is that the for loops are only executed a defined number of times.

When you are working with loops there are two very important statements that you should know.

Break statement: The break statement leaves the loop.

Continue statement: The continue statement returns to the beginning of the loop.
