Lists

In this chapter we'll be covering lists in Python.


1.1 Concept

Lists come in very handy when you're working with large chunks of data.

A list is essentially made up of one value: the list itself.

The list on the other hand has several values which are ordered and changeable.

The values in the list are called elements.

Lists begin and end with square brackets.

Every time you assign a list to a variable the elements in the list receive a value.

The leftmost element receives the index value zero and the rightmost element receives a positive index value depending on the number of elements in the list.

The values indicate the position of the element in the list and are called indices.

Indices start and end with square brackets just like lists do.

With the positive index you can now retrieve a specific element from the list.

It works like this: name_l[i_num].

Example: list_a[0] ⇨ will get you the first element(leftmost) of list_a.

It is also possible to retrieve an element from a list with a negative index.

The rightmost element receives the index value minus 1 and the leftmost element receives a negative index value depending on the number of elements in the list.

Example: list_a[-1] ⇨ will get you the last element(rightmost) of list_a.

Remember: The indices can only be integers! Using a different type will throw an error!

Remember: Using an index number that surpasses the number of available indices in the list will throw an error!


1.2 Manipulation

Slicing allows you to retrieve more than just an element from a list.

With an index value you can only retrieve one element from a list.

A slice also begins and ends with square brackets.

Slices are written as follows: name_l[i_n:i_n]

Example: list_a [0:3] ⇨ will get me the first three elements of list_a.

Remember: The third index will not be retrieved! Only index 0,1 and 2 will be returned.

If you want to change an already existing element in your list, then you can do this by referring to the element and adding the new value to this element.

Example: list_a[1]='Linus' ⇨ will change the second element in list_a to 'Linus'.

If you want to delete an element from a list, then you need to put del in front of the element that you want to delete.

Example: del list_a[1] ⇨ will delete the second element of list_a.

When working with lists you can also use for-loops!

You've probably heard of this concept in the conditions chapter, but I'll explain it again.

With the for-loop you can repeat everything inside the for-loop a defined number of times.

Example: for x in range (5):print(x) ⇨ will print: 0,1,2,3,4 (5 in total)

It is also possible to iterate (loop over) every element in a list.

Example: for x in list_a: print(x) ⇨ will print every element in list_a.

Furthermore you can use the operators in and not in to check, if an element is a list or not.

Example: if 'Linus' in list_a: print('He's there) ⇨ it will print the print-statement, if the condition is true.

This example can also applied to the operator not in to see, if 'Linus' is not in the list.


1.3 List vs Dict

Dictionaries are not sorted, hence dictionaries have no indices. That means that you can't retrieve one or more "values" with indices.

Lists on the other hand have indices that allow elements to be retrieved with indices.

Furthermore the order of the key-value pairs is not important, if you want to compare two lists for parity.

This is not the case when you're working with lists.


1.4 Methods

Index: The index method allows you to search for the index value of an element. If the element is not in the list, then Python will throw an error!

Remember: If there are several same elements in a list, then the first occurring element will be returned.

Append: The append method allows you to add an element to the end of the list.

Insert: The insert method allows you to add an element to a specific place in the list.

The insert method takes two arguments: the index value indicating the place where you want to add the value to the list and what you want to add to the list.

Remember: Append adds the element to the end of the list, whereas insert adds the element to a specific place in the list.

Remove: The remove method allows you to remove an element. It is also possible to remove an element with del, as we learned at the beginning of this chapter.

Sort: The sort method allows you to sort your list.
