1.2 Data Types

Every value in Python has a data type.

The most common data types are: Integers, Float and Strings.

You can recognize an integer by it's whole number (for example 3).

On the other hand a float number can have decimal places (for example 3.14).

Strings are characterized by their quotation marks (for example 'Linus').

Remember: An integer can never have decimal places!


1.3 Concatenation

Another important feature in Python is that strings can be concatenated.

Example: 'Linus' + 'Kenneth' = LinusKenneth.

Remember: One String can not be concatenated with an integer value. This will throw an error!


1.4 Variables

Variables are used to store values in them which then are used for further processing.

You assign a value to a variable with the equal sign.

Example: Linus = 22. I assigned 22 to the variable called "Linus".

Remember: Variables can only consist of letters, numbers and underscores!
