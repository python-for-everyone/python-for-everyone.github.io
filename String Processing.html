String Processing

In this chapter we will be covering useful methods and tricks when working with strings.


1.1 Methods

1.1.1 Boolean

isalpha() will return true, if a string only consists of letters.

isalnum() will return true, if a string only consists of letters and numbers.

isdecimal() will return true, if a string only consists of numeric characters.

These are not the only methods that will return a boolean value.

The method islower() will return true, if all letters in the string are small.

On the other hand, the method isupper() will return true, if all letters in the string are capitalized.

The startswith() method will return true, if the referred string startswith the characters in the brackets startswith() method.

The exact opposite can be done with the endswith () method.


1.1.2 Other

With the split method() you can split a string based on the argument of your method.

The join method allows you to a join string based on the argument of your method.

The lower() method converts a string into small letters.

The upper() method converts a string into capitalized letters.


1.2 Cool tricks

If you want to comment out several lines of code, then you can add three quotation marks at the front and at the end of the code that you want to comment out.

With the double quotation marks you can solve the problem of adding a "s" to the end of a word.

If you wouldn't put double quotation marks at the front and end of your string, then Python would throw an error!
