Classes

Today's topic are Classes in Python.

This is a very important concept in Python.

Mastering this concept will make your way of programming much more efficient.

1.1 Concept

Class: A class is a portrayal of an object from the real world.

Example: A class can be called "Animal". Animals exist in the real world.


Attributes: Attributes describe the class and the objects bound to the class.

Attributes: Number of legs, color of eyes and skin color, if we have a class called "Animal".


Objects: Objects are the concrete characteristics of the attributes.

In the class Animal we can have objects like: Dog, Cat and Rabbit.

These three objects or instances (both the same) are animals and are bound to the class animal.


Methods: Methods manipulate the attribute value(s) of an object.

Methods: get_n_o_legs, get_skin_col, get_color_eyes, if we a class called "Animal".


1.2 Classes

  • Define your class with def. In our case that is "Animal"
  • Define a constructor for your class
    • The constructor receives the arguments of the objects
    • In our case the constructor should include the attributes for our class
    • You define a constructor with def __init__
  • Define (self.xy = attribute) for every attributes in the class
    • self is just referring to the object that's calling the method
    • (self.xy = attribute) is essentially a variable for storing purposes
  • Define the methods: get_n_o_legs, get_skin_col, get_color_eyes
  • Instantiate objects for this class
