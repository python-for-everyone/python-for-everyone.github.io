Dictionaries

Today I would like to give an overview of dictionaries in Python.

Dictionaries are a very important concept in Python and will save you some time while coding.


1.1 Concept

This chapter will be shorter than the rest of the chapters, but don't underestimate the importance of dictionaries.

Dictionaries consist of keys and values. For that reason dictionaries are called key-value pairs.

The indices (the values in the dictionary) can be of any data type (integers,strings and floats).

Dictionaries are mainly used to organize the data and thereupon access this data.


1.2 Methods

Method is synonymous to function. Please don't get confused, because of the "unfamiliar" word!

If you remember what we did in chapter conditions, then this will sound familiar to you.

With the for loop you can iterate (loop over) values in the dictionary.

The for loop with the method values() allows you to iterate over all the values in the dictionary.

On the other hand the for loop with the method keys() allows you to iterate over all the keys in the dictionary.

Last but not least the for loop with the method items () delivers all the key-value pairs in the dictionary.

Those were three very important methods when it comes dictionary, but there are two further important methods.

The get() method takes two arguments: the key that you are searching for and a value that should be returned if the key doesn't exist.

The get() comes in handy, if you have a long dictionary and you're looking for a certain key.

The last method which I'll be explaining is the setdefault().

The setdefault method is similar to the get method.

Setdefault takes two arguments: the key that you are looking for and the value you want to set, if the key doesn't already exist.

If the key already exists, then setdefault will return the value for the respective key.


1.3 Manipulation

You can add a key-value pair to a dictionary by following this system: name_d["name_k"] = "name_v".

You can delete a key-value from dictionary by following this system: del name_d["name_k"].
